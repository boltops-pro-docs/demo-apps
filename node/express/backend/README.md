## Usage

Build and run

    npm install
    npm run server # nodemon app.js

For deployment

    npm run start  # node app.js

Curl test

    curl localhost:3000

## Notes

To build the initial app

    npm init
    npm install --save express
    npm install --save morgan
    npm install --save nodeman
